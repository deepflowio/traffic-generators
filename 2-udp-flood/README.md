# Build

sh ./build.sh

# Run

```
Usage of ./udp_flood:
  -h string
        Target host (default "localhost")
  -p int
        Target port (default 12345)
  -r int
        Packets per second (default 100000)
  -s int
        Data size (default 16)
  -t int
        Number of threads (default 1)
```
